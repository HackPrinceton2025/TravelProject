# TravelProject Agent

This directory contains:

- prompts/ system prompts for Claude and Grok
- tools/ JSON Schemas for tools used by the agent
- dedalus_client.py minimal HTTP client for Dedalus-compatible API

Configure environment via root or backend `.env`:

- DEDALUS_API_BASE
- DEDALUS_API_KEY
- ANTHROPIC_MODEL
- XAI_MODEL
